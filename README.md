# ⚡ What's Your Superpower? — NXT WAVE Quiz

A mobile-first interactive quiz that helps people discover their gifting category: **Creator**, **Problem-Solver**, **Protector**, or **Communicator**.

Built for NXT WAVE's in-person event where attendees dress in their assigned color and bring something that represents their gift.

## How it works

- 10 multiple-choice questions
- Automatic scoring with tiebreaker logic
- Result screen with category, color code, dress code instructions, and a Bible verse
- Optimized for mobile screenshots

## Categories & Colors

| Category | Color |
|---|---|
| 🎨 Creator | Red |
| 🧠 Problem-Solver | Blue |
| 🛡️ Protector | Yellow |
| 📢 Communicator | Green |

## Tech

Single HTML file, zero dependencies, no backend. All data stays on the user's device — nothing is stored or collected.
